Generates a program to compute the factorial of a number using a custom instruction set.

This function creates a sequence of instructions that calculates the factorial of a number stored at a specific memory address (input_addr) and stores the result at another memory address (result_addr). The program uses a loop to multiply numbers from the input value down to 1, storing the intermediate results in a register.

Parameters
input_addrThe memory address where the input number (n) is stored.
result_addrThe memory address where the computed factorial result will be stored.
Returns
A vector of instructions representing the factorial computation program.

Instructions:

  • Initializes registers:
    • R0: Stores the result (initialized to 1).
    • R1: Stores the input value (n).
    • R2: Constant value 1.
    • R3: Constant value 0.
  • Checks if the input value is 0 and directly stores 0 as the result if true.
  • Implements a loop to multiply the result by the current value of R1 (n) and decrement R1 until it reaches 1.
  • Stores the final result in memory at result_addr.
  • Halts the program.

Generates a program to compute the Fibonacci sequence up to a given number.

This function creates a sequence of instructions for a RISC-like emulator to compute the Fibonacci number for a given input n. The result is stored in the specified memory address.

Parameters
input_addrThe memory address where the input value n is stored.
result_addrThe memory address where the computed Fibonacci result will be stored.
Returns
A vector of instructions representing the program to compute the Fibonacci sequence.
  • If n == 0, the result is 0.
  • If n == 1, the result is 1.
  • For n > 1, the program iteratively computes the Fibonacci sequence using a loop.
  • The program handles potential overflow during addition and halts execution if detected.
  • The result is stored in the memory address specified by result_addr.
